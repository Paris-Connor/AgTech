Educational Drum Machine

Click steps to build a 16-step pattern. Learn how BPM maps to time, what “swing” does, and why scheduling audio with AudioContext is more reliable than setInterval.

Steps: 16
Grid: 4/4
Engine: Web Audio
Keys: Space Play
Kick
Snare
Hi-Hat
Playhead moves left → right each step
Step length: 125 ms • 16 steps = 2000 ms
Scheduling: lookahead 25 ms • schedule ahead 0.10 s

Waveform / Activity

AnalyserNode

If the scope is flat, press Play. If your browser blocks audio, click anywhere once to “unlock” audio.

What you’re learning

  • BPM → seconds: one beat = 60/BPM seconds. One 16-step bar in 4/4 = 4 beats.
  • Steps: 16 steps per bar → step duration = (60/BPM)/4.
  • Swing: delays every other 16th note to create a “lilt”.
  • Accurate timing: we schedule sounds slightly ahead using audioCtx.currentTime.
  • Synthesis: kick/snare/hat are generated (no samples) so it’s portable.

Keyboard shortcuts

  • Space play/stop
  • C clear pattern
  • R randomize
  • 1/2/3 load presets
Built with inline HTML/CSS/JS + Web Audio. Educational focus: timing, scheduling, synthesis.
